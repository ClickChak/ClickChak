1. dSploit

dSploit is a network penetration suite for Android that allows you to scan networks for connected devices, identify its operating system, run services, and open ports in each device. You can also use this to check for vulnerabilities in your network and perform "man in the middle" operations. To perform these operations, dSploit offers a Wi-Fi scanning feature that conveniently scans for keys and other supported models.

OS: Android

Root: Yes

Size: 6.5MB

2. Zanti

zANTI is a mobile penetration testing and security analysis tool that lets you see how risky a network actually is. Using zANTI allows you to simulate a cyber attack—specifically Man In the Middle (MITM) attacks—to test networks and identify vulnerabilities in them. This tool will let you see the users connected to the network, the weak points in their devices and the network itself, and where you should apply further cyber-protection. zANTI is fast in its work and offers a user-friendly interface that you can master in no time.

OS: Android

Root: Yes

Size: 23.9MB

3. Kali Nethunter

Kali NetHunter is a free & Open-source Mobile Penetration Testing Platform for Android devices, based on Kali Linux.

OS: Android

Root: No/ Yes

Size: 76.7MB
